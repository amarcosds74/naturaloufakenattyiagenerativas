# naturaloufakenattyiagenerativas
# Natural ou Fake Natty? Como Vencer na Era das IAs Generativas


**Autor:** Antonio Marcos Dos Santos


**Descrição:** E-book que explora o impacto das IAs generativas na autenticidade pessoal e profissional, propondo estratégias éticas e práticas para vencer na era da IA.


## Conteúdo
- Introdução
- A ascensão das IAs generativas
- "Suplementos digitais"
- Produtividade com IA
- Fake Natty Ético
- Profissões e futuro do trabalho
- Habilidades para vencer na era da IA
- Conclusão


## Arquivos incluídos
- `src/roteiro.md` — texto completo do e-book em Markdown
- `output/ebook.pdf` — versão em PDF pronta para download
- `src/images/capa.png` — (opcional) capa do e-book


## Como usar
1. Baixe/clone este repositório.
2. Verifique `output/ebook.pdf`.
3. Para editar o texto, abra `src/roteiro.md`.
4. Se quiser gerar novo PDF, use o arquivo Markdown e qualquer conversor (Pandoc, Canva, Google Docs).
